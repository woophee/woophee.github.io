hexo常用指令

发表于 2018-09-16 | 更新于 2018-10-23 | 分类于 Hexo | 评论数：

新建文章

hexo new "postName" #好处是自动生成日期

新建页面

hexo new page "pageName"

生成静态页面至public目录

hexo generate

开启预览访问端口

hexo server #默认端口4000，'ctrl + c'关闭server

部署到GitHub

hexo deploy

查看帮助

hexo help

查看Hexo的版本

hexo version

新建页面

hexo clean #清除缓存文件db.json和已生成的静态文件public

缩写：

hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy

组合命令：

hexo s -g #生成并本地预览
hexo d -g #生成并上传

备注

hexo s -g 本地启动localhost:4000端口，4000端口可能被占用
hexo s -g –p 5000 本地改启动5000端口启动服务
