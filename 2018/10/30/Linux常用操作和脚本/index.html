添加互信

可以设置自己和自己免密

机器列表(list)：

IP         账号    密码    
1.1.1.1    root    mima1
2.2.2.2    root    mima2
3.3.3.3    root    mima3

互信脚本(trust.sh)：

#!/bin/sh

INVENTORY_LIST=$1

while read line;
    do
    USER=$(echo $line|tr -s " "|cut -d " " -f2)
    ADDR=$(echo $line|tr -s " "|cut -d " " -f1)
    PASSWORD=$(echo $line|tr -s " "|cut -d " " -f3)
    echo $USER;
    echo $ADDR;
    echo $PASSWORD;

expect -c "
    set timeout 10
    spawn ssh-copy-id -i /root/.ssh/id_rsa.pub $USER@$ADDR
    expect{
        \"*yes/no*\"{send \"yes\r\"}
        \"*password:*\"{send \"$PASSWORD\r\"}
    }
expect eof"
done < $INVENTORY_LIST

执行：

sh trust.sh list
